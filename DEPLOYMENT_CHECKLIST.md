## 🔧 常见问题排查

### 如果发现问题，按以下步骤排查：

1. **资源加载失败**
   - 检查浏览器控制台 Network 标签
   - 确认 URL 是否正确（应该包含 `/3DUniverse/` 前缀）
   - 检查文件是否在 `public/` 目录下

2. **音频无法播放**
   - 确认已经与页面交互（点击过）
   - 检查浏览器是否阻止自动播放
   - 查看控制台是否有错误

3. **页面空白**
   - 检查控制台错误信息
   - 确认 JavaScript 是否加载
   - 尝试清除缓存刷新

4. **性能问题**
   - 降低时间速度
   - 关闭浏览器其他标签
   - 检查设备性能
