# 音频文件说明

## 音乐文件夹
放置位置：`public/music/`

### background.mp3
- 太空背景音乐
- 推荐风格：舒缓的太空氛围音乐
- 格式：MP3
- 时长：建议 2-5 分钟循环

## 音效文件夹
放置位置：`public/sounds/`

### jump.mp3
- 相机跳转音效
- 推荐风格：科幻风格的速度感音效
- 格式：MP3
- 时长：0.5-1 秒

### click.mp3
- 点击音效
- 推荐风格：清脆的点击声
- 格式：MP3
- 时长：0.1-0.3 秒

### hover.mp3
- 悬停音效
- 推荐风格：轻微的提示音
- 格式：MP3
- 时长：0.1-0.2 秒
